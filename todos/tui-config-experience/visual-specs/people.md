# People Config Spec

## Animation Concept

- **Idle:** A generic user icon standing still.
- **Interaction:** The user icon waving or moving.
- **Success:** The user icon gives a thumbs up.
- **Error:** The user icon turns red/crosses arms.

## Color Palette

- **Primary:** #FFFFFF (Curses: WHITE)
- **Secondary:** #CCCCCC (Curses: GREY)
- **Background:** Default terminal background

## ASCII Assets

- User: `(@)`
