# AI Keys Config Spec

## Animation Concept

- **Idle:** Digital rain (matrix style) falling slowly.
- **Interaction:** Keys glowing brighter as typing occurs.
- **Success:** A lock icon opening.
- **Error:** A lock icon shaking violently.

## Color Palette

- **Primary:** #00FF00 (Curses: GREEN)
- **Secondary:** #FFD700 (Curses: YELLOW/GOLD)
- **Background:** Default terminal background

## ASCII Assets

- Lock: `[#]` -> `[ ]`
- Rain: `1 0 1`
