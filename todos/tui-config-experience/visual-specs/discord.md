# Discord Config Spec

## Animation Concept

- **Idle:** A blurple pulse emanating from the logo.
- **Interaction:** Typing dots (...) appearing and disappearing.
- **Success:** A green checkmark appears.
- **Error:** A red X flashes.

## Color Palette

- **Primary:** #5865F2 (Curses: INDIGO/BLUE)
- **Secondary:** #FFFFFF (Curses: WHITE)
- **Background:** Default terminal background
