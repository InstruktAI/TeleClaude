# Notifications Config Spec

## Animation Concept

- **Idle:** A bell icon gently swaying.
- **Interaction:** The bell ringing (sound waves).
- **Success:** The bell glowing.
- **Error:** The bell cracked.

## Color Palette

- **Primary:** #FF9900 (Curses: YELLOW/ORANGE)
- **Secondary:** #FFFFFF (Curses: WHITE)
- **Background:** Default terminal background

## ASCII Assets

- Bell: `/o`
