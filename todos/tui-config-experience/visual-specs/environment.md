# Environment Config Spec

## Animation Concept

- **Idle:** A globe spinning slowly.
- **Interaction:** Clouds drifting by.
- **Success:** Sun rising.
- **Error:** Storm clouds.

## Color Palette

- **Primary:** #00FF00 (Curses: GREEN)
- **Secondary:** #00CCFF (Curses: CYAN)
- **Background:** Default terminal background

## ASCII Assets

- Globe: `(O)`
