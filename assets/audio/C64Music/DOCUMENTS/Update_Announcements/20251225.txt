# cRSID 1.56 #
##############

cRSID is an open-source cycle based sidplayer made by Mihaly Horvath (Hermit).
This is an integer-only rewrite of cSID and cSID-light. The new sidplayer is
able to play most RSID tunes and of course PSID tunes. It includes a
pre-compiled executable for Windows and Linux.

Changes since version 1.4:

- Fixed broken OSC3/ENV3 readout caused by the 1.55 code overhaul
- Integrated direct ALSA audio backend into libcRSID
- Added new crsid_cli command-line-only version
- Embedded a heavily stripped-down SDL 1.2 version into the application
- Reworked CLI keyboard handling
- Major audio quality upgrade
  - Higher-quality resampling of oversampled SID emulation output
  - Further optimized for performance and power efficiency
- Major internal refactor and optimization pass
- Significant performance improvements, especially on low-power hardware
- Improved RSID compatibility
- Added file browser with GUI-based playlist support
- Support for loadable ROM files
- Source code refactoring and cleanup
- Enhanced filter distortion control
- Dropped support for non-Linux platforms
- Documentation update
- Bug fixes and stability improvements

For more details go to:

https://csdb.dk/release/?id=257992

# SID2TXT v1.0 #
################

Stello Doussis released a new online tool called 'SID2TXT' for converting SID
tunes into a clean and customisable text log.
SID2TXT emulates a C64 and plays SID files silently, generating a frame-by-frame
text log of SID register activity ($D400-$D418). It supports PSID and RSID
files and is useful for studying drivers, analyzing music techniques or
debugging code.

Check it out:

https://www.doussis.com/sid2txt/

# SID Known v1.30 #
###################

Wilfred Bos has released a new version of SID Known. This version supports now
HVSC #84. SID Known is a command line tool which you can use to identify SID
tunes from SID and PRG files.

This tool can be used if e.g., you want to know which SID tune is used in a
specific C64 demo or C64 game, or you have a SID tune found or ripped and you
want to know if it is already in your SID collection.

SID Known can be used to identify music but it also helps to find out which
tunes are not in the High Voltage SID Collection. If you find music not found
by SID Known, it would be appreciated if you report this to the HVSC crew
(https://hvsc.de) or post it in one of the CSDb forums (https://csdb.dk) like
"SIDs that aren't in HVSC yet".

Get the tool here: https://csdb.dk/release/?id=258064
