Reference
- /new_session: create a new tmux session.
- /agent_restart: restart the active agent in a session.
- /agent_resume: resume the last agent for a session.
- /claude, /gemini, /codex: start the named agent.
- /claude_plan: navigate to Claude plan mode.
- /help: show command help.
- /cancel, /cancel2x, /kill: send control signals.
- /tab, /shift_tab, /enter, /escape, /escape2x, /backspace.
- /key_up, /key_down, /key_left, /key_right.
- /ctrl: send CTRL+key.
