Standard
- Follow coding/testing directives and existing project patterns.
- Add/update unit or integration tests for changes and run make test.
- Keep secrets in .env and avoid duplicating teleclaude.db in the main repo.
- Maintain clean UX message handling and command registration rules.
- Summarize changes, tests, and any deferrals for handoff.
