# Use Cases

## Human Use Cases (Scrum Style)

1) **Create an empty session**  
As a user, I want to create an empty terminal session so I can work manually in a custom folder.

2) **Create a session with an agent**  
As a user, I want to start a session with an agent and a thinking mode so I can begin collaborating immediately.

3) **Create a session with agent + initial task**  
As a user, I want to start a session with an agent and provide an initial instruction so the agent begins working right away.

4) **Resume an existing agent session**  
As a user, I want to resume a previous agent session by its native session id so I can continue where I left off.

5) **Restart an agent in a session**  
As a user, I want to restart the agent in a session when it’s stuck or unresponsive.

6) **Send a command/message to a session**  
As a user, I want to send commands or messages to an existing session so I can drive the terminal or agent.

7) **View live output from a session**  
As a user, I want to see live session output so I can monitor progress in real time.

8) **Attach to a session in the TUI**  
As a user, I want to open a session’s tmux pane inside the TUI so I can interact directly.

9) **End a session**  
As a user, I want to end a session so resources are cleaned up and it disappears from my list.
