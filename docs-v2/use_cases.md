# Use Cases

## Human Use Cases (Scrum Style)

1) **Create an empty session (escape hatch)**  
As a user, I want to create an empty terminal session so I can work manually in a custom folder when no project mapping exists.

2) **Create a session with an agent**  
As a user, I want to start a session with an agent and a thinking mode so I can begin collaborating immediately.

3) **Create a session with agent + initial task**  
As a user, I want to start a session with an agent and provide an initial instruction so the agent begins working right away.

4) **Resume an existing agent session**  
As a user, I want to resume a previous agent session by its native session id so I can continue where I left off.

5) **Restart an agent in a session**  
As a user, I want to restart the agent in a session when itâ€™s stuck or unresponsive.

6) **Send a command to a session**  
As a user, I want to send a command to a session so the agent CLI executes it.

7) **Send a message to a session**  
As a user, I want to send a plain-text message to a session so the agent receives guidance.

8) **View live output from a session**  
As a user, I want to see live session output so I can monitor progress in real time.
 
9) **End (close) a session**  
As a user, I want to end a session so resources are cleaned up and the session is marked closed.
